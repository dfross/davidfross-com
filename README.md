# DavidFross.com

This repository hosts the source code for my personal website, [davidfross.com](https://davidfross.com).

## Overview

I built this site from the ground up to be exceptionally fast, accessible, and visually engaging. It acts as a small, polished showcase of modern web performance and technical design.

**Highlights:**

- ⚡ **Lightweight & Fast:** Statically generated via Astro and aggressively optimized.
- 🎨 **Kinetic WebGL:** Features a custom, interactive 3D WebGL shader background.
- ♿ **Accessible:** Built with `prefers-reduced-motion` support, screen-reader text, and semantic HTML.

## Tech Stack

- **Framework:** [Astro](https://astro.build/)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
- **Interactivity:** Vanilla TypeScript & WebGL
- **Deployment:** Netlify
